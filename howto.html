BASIC TEXT CONVERSION

Text Tab

Copy and paste your translated chapter into the left-hand box with the tabs on top. The dialogue should be formatted like this:
Person A: How are you today?
Person B: I'm crying about enstars, you?
Person A: Lol same

Details Tab

Fill in information about the chapter and its appearance on the wiki. INSERT DIAGRAM OF WHAT EACH COLOR IS

Renders Tab

The renders tab should automatically display which characters are in the dialogue once you paste it into the text tab. Fill in the file names of the renders that should be used for each character. (ex. Tsukasa Suou School Dialogue Render.png)
NOTE: For now, if you want to use more than one render per character in one chapter, you have to format each section of the chapter with different renders separately. INSERT EXAMPLE

TL Notes Tab

If the chapter has translation notes, insert a number for each note into the dialogue like so:
Person A: How are you today? [1]
Person B: I'm crying about enstars, you? [2]

Then paste the list of translation notes into the TL Notes tab:

[1] A question commonly used to start conversation.
[2] Ensemble Stars is a popular mobile game that causes many tears.

NOTE: It's important to separate each note with a line break.
NOTE: To have links in your translation notes, make sure to format them like this: [https://www.example.com text that is link]

CREATING A NEW PAGE ON THE WIKI

  1. Copy the text that shows up on the right-hand box (can be done with Ctrl+A/Cmd+A)
  2. Go to the cover page of the story and edit the URL to have the name of the chapter.
    For example, https://www.ensemble-stars.wikia.com/wiki/Checkmate/CHAPTER TITLE
  3. Choose to create a new page.
  4. Instead of the visual editor, switch to the source editor.
  5. Paste in the formatted text, and publish the page.
  6. NOTE: As of now you must manually upload the header image of the chapter in the visual editor unless you upload the header image to the wiki before creating the new chapter page.
