☆ENSTARS STORY FORMATTER☆

A website to more easily upload event/gacha stories from the mobile idol game Ensemble Stars to the fandom wiki.
It takes translated story dialogue and formats it into text that you can paste directly into the "source" section of the page.
Developed by gayandasleep@twitter.

  • HOW TO USE
  • QUESTIONS/FEEDBACK
  • DESIGN PLANS

Translator notes

  • How are translator notes typically represented in the dialogue?
  • Most common format I've seen is in the relevant dialogue section in italics
  • Is current method of using numbered markers to insert tl notes easy to use/visually appealing
  • How to deal with links in tl notes?

Choosing renders

  • Instead of pasting in name of render file, have a dropdown list with all the possible renders for that character
  • Would require somehow pulling that list from the wiki
  • What about renders that haven't been uploaded yet?

Formatting CGs and scene changes

  • People typically insert new rows for when the story displays CGs and scene changes
  • Do scene changes typically have just a row with text stating Scene Change: (NEW LOCATION) or do they also put in a new screenshot of the in-game story? Which looks better?
  • Having more markers seems cumbersome so possible solution:
  • Users place the cursor where the CG/scene change should be in the dialogue
  • New tab in input area that allows user to choose whether CG or scene change should be inserted
  • Input field for relevant wiki file/new location
  • Formatter inserts marker in dialogue, edit backend to handle marker
  • Could alternatively be a tab in output area that inserts wiki code directly

Working in smaller windows

  • Make formatter more presentable for windows that aren't fullscreen
  • People might have formatter open in one half-screen window and the wiki open in another half-screen window
  • Is this necessary/needed?

Visual design
